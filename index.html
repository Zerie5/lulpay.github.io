About LulPay

LulPay is a modern mobile wallet application that allows users to manage multiple currencies, send and receive money, and make payments using QR codes. Built with Flutter, it provides a seamless experience across both iOS and Android platforms.

Key features include:

  • Multi-currency wallet management
  • Secure money transfers
  • QR code payment system
  • Real-time currency updates
  • User-friendly interface
