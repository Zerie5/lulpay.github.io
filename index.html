About LulPay

LulPay is a comprehensive mobile wallet solution that allows users to manage multiple currencies, send and receive money, and make payments using QR codes. The system consists of a Flutter-based mobile application for iOS and Android, backed by a robust Spring Boot backend service.

Key features include:

  • Multi-currency wallet management
  • Secure money transfers
  • QR code payment system
  • Real-time currency updates
  • User-friendly interface
  • Non-wallet transfers to unregistered users
  • Multi-channel notifications (Push, SMS, Email)

Frontend (Flutter)

The LulPay mobile application is built using:

  • Flutter: Cross-platform UI framework
  • GetX: State management and routing
  • RESTful APIs: For backend communication

The mobile app provides a seamless user experience across both iOS and Android platforms with a modern, intuitive interface.

Backend (Spring Boot)

The LulPay backend service is built using:

  • Java 17: Core programming language
  • Spring Boot 3.x: Application framework
  • Spring Data JPA: Data access layer
  • PostgreSQL: Relational database
  • Firebase Admin SDK: For push notifications

The backend provides a secure, scalable API for all mobile app operations with robust transaction processing.
