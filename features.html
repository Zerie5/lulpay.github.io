Multi-Currency Wallet

The wallet is the core feature of LulPay, allowing users to manage multiple currencies in one place.

Key Capabilities:

  • View total portfolio value across all currencies
  • Track individual currency balances
  • Add new currencies to your wallet
  • View transaction history for each currency

Send Money

The send money feature allows users to transfer funds quickly and securely.

Key Capabilities:

  • Send money to other LulPay users
  • Transfer between your own accounts
  • Schedule recurring transfers
  • View transfer history

Receive Money

Users can receive money through multiple convenient methods.

Key Capabilities:

  • Generate a QR code for receiving payments
  • Share your unique user ID
  • Create payment requests
  • Receive notifications for incoming transfers

QR Code Payments

Make quick and easy payments using QR code scanning.

Key Capabilities:

  • Scan QR codes to make payments
  • Generate QR codes for receiving payments
  • Secure transaction processing
  • Transaction history and receipts

User Profile

Manage your user profile and account settings.

Key Capabilities:

  • View and update personal information
  • Manage security settings
  • Set language preferences
  • Configure notification preferences
